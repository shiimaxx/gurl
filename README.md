# gurl

gurl is a like curl command by go

## Usage

```
gurl [(-p | --parallel) <parallel>] (-o | --output) <file> <url>
```

```
Usage of gurl:
  -o string
        output file(Short)
  -output string
        output file
  -p int
        number of parallel(Short) (default 10)
  -parallel int
        number of parallel (default 10)
  -version
        print version information
```
